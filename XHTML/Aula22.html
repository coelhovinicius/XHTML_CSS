!DOCTYPE

!DOCTYPE é um dos itens obrigatórios em uma página em XHTML

!DOCTYPE é o conjunto de definições para o tipo de documento a ser construído, no caso, XHTML. Informa as regras para validar os documentos (páginas).

Para estruturar uma página em XHTML, o !DOCTYPE é obrigatório. Abaixo está um exemplo do mínimo necessário para se estruturar uma página na Internet.

		---AS TAGS FORAM SUBSTITUÍDAS POR ASTERÍSCOS---
		
		*!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
		"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"*

		*html xmlns="http://www.w3.org/1999/xhtml"*

		*head*
		  *title*Title of document*/title*
		*/head*

		*body*
		  some content
		*/body*

		*/html*

DTD

DTD é um dos dados informados no !DOCTYPE. DTD significa "Document Type Definition". É um arquivo onde colocamos as regras de como construir bem o documento, terá as regras de validação da página.

Há três tipos de DTDs:

  • STRICT - Para desenvolvedores profissionais, pois há padrões rígidos de controle de construção da página. Só permite definição de estilos com o uso do CSS. This DTD contains all HTML elements and attributes, but does NOT INCLUDE presentational or deprecated elements (like font). Framesets are not allowed.
  • TRANSITIONAL - Um pouco mais flexível. Permite definição de estilos sem CSS. This DTD contains all HTML elements and attributes, INCLUDING presentational and deprecated elements (like font). Framesets are not allowed.
  • FRAMESET - Não é considerado ser utilizado na construção das páginas, pois dificulta a acessibilidade, por exemplo. This DTD is equal to Transitional, but allows the use of frameset content.

STRICT - XHTML 1.0 - EXEMPLO:

		---AS TAGS FORAM SUBSTITUÍDAS POR ASTERÍSCOS---
		
		*!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
		"http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd"*

TRANSITIONAL - XHTML 1.0 - EXEMPLO:

		---AS TAGS FORAM SUBSTITUÍDAS POR ASTERÍSCOS---
		
		*!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
		"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"*

FRAMESET - XHTML 1.0 - EXEMPLO:

		---AS TAGS FORAM SUBSTITUÍDAS POR ASTERÍSCOS---
		
		*!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Frameset//EN"
		"http://www.w3.org/TR/xhtml1/DTD/xhtml1-frameset.dtd"*
