Elementos Inline - Aula 04

Elementos que alteram o texto na própria linha

Por exemplo, conseguimos colocar apenas a palavra "negrito" em negrito com a tag "b".
Para Itálico, temos a Tag "i". Agora, esse texto ficou em Itálico, e o NEGRITO está em negrito.
Há Tags que estão caindo em desuso, como a "u", de "underline". Há uma substituição dessa Tag, usando CSS.
Temos a Tag "del", que demonstra que algum trecho do texto pode estar prestes a ser excluído como, por exemplo, essa parte nada.
A Tag Teletipo faz com que todas as letras das palavras ocupem o mesmo espaço na tela.
Tag "sup": faz com que o texto fique sobrescrito. "Sub", faz o texto ficar subescrito.

Podemos dar ênfase a trechos do texto, informando ao browser que a parte entre a Tag específica é de maior importância. Trata-se da Tag "em".
Para ainda mais ênfase, utilizamos a Tag "strong".
As Tags acima, não apenas efatizam visualmente o texto, como também informa ao browser que essas partes específicas do texto são de maior importância e merecem maior atenção. Essas Tags não servem para estilizar o texto, elas informam o que cada parte do texto significa, como Título, Subtítulo, Parágrafos, Linhas, entre outros.
Outra Tag informa ao browser que o trecho do texto é uma citação, como Ser ou não ser, eis a questão. Essa tag "q" impõe aspas ao trecho selecionado.
A Tag "big" aumenta um pouco o trecho do texto e a Tag "small" diminui um pouco.
A Tag "marquee" está depreciada também, substituída pela CSS. Para maiores informações em Tags HTML, temos os sites www.codex.wiki.br/Html/ListaDeTags e www.w3c.org, onde há referências da linguagem e elementos do HTML.
