TABELAS - TABLE, TR e TD - INTRODUÇÃO

Podemos criar tabelas dentro da página. Como explicado na aula anterior, as tabelas devem ser utilizadas, apenas, para listar algo, itens.
Para cada linha da tabela, utilizamos a Tag TR - Table Row. Para cada coluna, a Tag TD - Table Data.
Abaixo, podemos ver o esqueleto de uma tabela com quatro linhas e duas colunas (substitua os pontos por Tags):

		.table border border="1" cellpadding="10" (indica o espaço entre o conteúdo e a borda da célula) cellspacing="0" (indica o espaço entre as células).
			.tr..td.TREINAMENTO./td..td.INSTRUTOR./td../tr.
			.tr..td.Scrum./td..td.Aman./td../tr.
			.tr..td.Lumis./td..td.Josadache./td../tr.
			.tr..td.HTML./td..td.Esaú./td../tr.
		./table.

Ainda temos atributos que podemos utilizar dentro da TD. Dentro da Tag TD, podemos utilizar o atributo align="right" (left, center, justify).
Podemos formatar vários estilos, como cor do fundo, cor da fonte, etc..
