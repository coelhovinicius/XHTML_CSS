Aula 05 - Atributo "style"

Atributo Inline que altera as propriedades de um trecho do texto

Para isso, utilizamos a propriedade "color".
Há três formas de atribuirmos a cor: pelo nome em inglês, pelo padrão RGB decimal ou pelo padrão RGB hexadecimal, um padrão das cores vermelho, verde e azul.
Para o padrão hexadecimal, utilizamos utilizamos a hashtag #. Por exemplo, #FF0000, vermelho.
Para o decimal, utiliza-se style="color:rgb(200,200,23)".
O hexadecimal é o mais utilizado.

Para maiores informações, há o site http://erikasarti.net/html/tabela-cores/.
