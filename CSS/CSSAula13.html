BACKGROUND

O Background é uma propriedade que permite adicionar efeitos ao Box Model de qualquer elemento da página XHTML. Essa propriedade não é herdada.
O valor padrão de Background é transparente. Caso não seja aplicado efeito de Background a um elemento, essa propriedade irá assumir o valor do elemento pai, porém, não por herança e, sim, pelo fato do valor padrão ser transparente. Então, o que houver no conteúdo do Background do elemento pai será mostrado, também, no elemento filho.
Há cinco tipos para o Background:

  • Color - background-color:#00BFFF; - altera a cor de fundo
  • Image - background-image:url("../img/exemplo.png") - atribui uma imagem ao fundo
  • Repeat - background-repeat:no-repeat; (ou "repeat-x" ou "repeat-y") - indica se a imagem será repetida. Por padrão, a imagem é repetida, mesmo sem colocar "repeat"
  • Attachment - background-attachment:fixed; (ou scroll) - indica o alinhamento da imagem do elemento em relação às demais imagens da página
  • Position - background-position:top right; (e outros diversos exemplos) - define um posicionamento preciso para a imagem dentro do elemento

Background Color

Como já visto em XHTML, essa propriedade altera a cor do fundo do elemento a ser customizado. Podemos atribuir uma cor qualquer, ou podemos atribuir a cor padrão da janela ativa, utilizando o ActiveBorder: 

				p{
					background-color:ActiveBorder;
				}

Background Image

Podemos adicionar uma imagem como plano de fundo utilizando a propriedade background-image. Utilizamos a seguinte sintaxe "background-image:url();". 

				p{
					background-image:url("../img/exemplo.png");
				}

Background Repeat

Informa se queremos que a imagem se repita. Por padrão, ela será repetida dentro do Box Model ao qual estamos aplicando a imagem. Temos:

  • background-repeat:no-repeat; - informa que não queremos que a imagem seja repetida no elemento
  • background-repeat:repeat-y; - repete a imagem apenas na vertical - eixo y
  • background-repeat:repeat-x; - repete a imagem apenas na horizontal - eixo x

Attachment

Utilizado para "alinhar" as imagens, por exemplo. Pode ser "fixed", que alinha ao fundo de outros elementos da página, ou "scroll", que alinha dentro do próprio elemento. 

				p{
					background-attachment:fixed;
				}

Background Position

Define a posição da imagem no elemento.

Por Definições Diretas

  • background-position:top left; - posicionamento padrão
  • background-position:top;
  • background-position:top right;
  • background-position:center left;
  • background-position:center;
  • background-position:center right;
  • background-position:bottom left;
  • background-position:bottom;
  • background-position:bottom rigth;

Por Valores Fixos

  • Pixels: background-position:20px 90px; - Distanciamento de 20px para o eixo "x" e 90px para o eixo "y"
  • Porcentagem: background-position:50% 50%; - distancia de 50% do valor

Podemos utilizar quaisquer unidades de medida, bem como valores negativos.

SIMPLIFICAÇÃO DO CÓDIGO PARA BACKGROUND

Podemos simplificar a declaração do Background utilizando a seguinte sintaxe: 

				div{
					background:#00BFFF url("../img/exemplo.png") no-repeat fixed 30% 30%;
				}
