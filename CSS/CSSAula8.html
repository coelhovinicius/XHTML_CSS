BOX MODEL

O Box Model nos possibilita determinar a maneira como os elementos são exibidos e pode determinar a maneira como esses elementos interagem entre si dentro de cada elemento em bloco da página a ser desenvolvida. Podemos customizar o "body", as "div", os "p", os "h1", "h2", etc..
Cada Box Model possui os elementos Margem, Borda, Preenchimento e Conteúdo, onde podemos customizar da maneira necessária. Podemos, por exemplo, mudar o tamanho, cor, layout desses elementos.
Para vizualização dos códigos do site, podemos utilizar, no Firefox, a ferramenta chamada Firebug.

EXEMPLO - Customização de Margem

				p{
					margin-top:1px;
					margin-right:50px;
					margin-bottom:50px;
					margin-left:20px;
				}

SIMPLIFICAÇÃO

Podemos simplificar o código para a customização das medidas da Margem. Para isso, devemos seguir a ordem de lados Topo, Direita, Baixo e Esquerda. 

				p{
					margin:1px 50px 50px 20px;
				}

Ainda, podemos simplificar mais. Quando atribuímos apenas um valor, este é aplicado a todos os lados: 

				p{
					margin:1px;
				}
				
				OU
				
				p{
					margin:1px 2px;
				}
				
				OU AINDA
				
				p{
					margin:1px 2px 3px;
				}

PADDING - Preenchimento

Para customizarmos a área do PREENCHIMENTO, utilizamos códigos semelhantes aos de cima, mudando "margin" para "padding", como visto abaixo: 

				div{
					padding:2px 5px;
				}

BORDA

A propriedade Border é um pouco mais complexa. Para customização dessa propriedade, podemos alterar os elementos Largura, Cor e Estilo de borda.
Como com os exemplos acima, há diversas maneiras de determinar o layout das bordas. Podemos alterar a largura e cor de todas as bordas de uma vez, ou podemos alterar cada lado da borda que queremos aplicar a alteração. 

				p{
					border-top-width:5px;
					border-right-width:10px;
					border-bottom-width:5px;
					border-left-width:10px;
					border-top-color:red;
					border-right-color:blue;
					border-bottom-color:green;
					border-left-color:yellow;
				}
				
				OU
				
				p{
					border-width:5px 10px 5px 10px;
					border-color:red blue green yellow;
				}
				
				OU
				
				p{
					border-width:5px;
					border-color:red;
				}
				
				OU
				
				p{
					border-width:5px 10px;
					border-color:green yellow;
				}
				
				OU
				
				p{
					border-width:5px 10px 20px;
					border-color:blue yellow green;
				}

Estilos de Bordas

Há diversos estilos de bordas que podem ser aplicados ao layout da página. Bordas sólidas, pontilhadas, tracejadas, em alto-relevo, etc.. 

				div{
					border-top-style:solid; - borda sólida
					border-right-style:dotted; - borda pontilhada
					border-bottom-style:ridge; - borda em alto-relevo
					border-left-style:dashed; - borda tracejada
					border-style:double; - borda dupla;
					border-style:inset; - borda "para baixo"
					border-style:outset; - borda "para cima"
				}

Ainda, podemos simplificar a sintaxe das bordas, como abaixo: 

				div{
					border:5px dashed purple; - assim, todos os lados receberão as mesmas características
				}
