Menu Drop Down com JQuery

OBSERVAÇÃO IMPORTANTE: Quando estamos programando, é melhor montarmos nossa folha de estilos direto no cabeçalho do arquivo XHTML. Com isso, ganhamos agilidade, bem como o próprio browser lê mais rápido sua págiina. Após estar tudo OK, cortamos o estilo e inserimos em um arquivo CSS.

Para a construção de menus padrão Web, Horizontais, Verticais ou Sobmenus, utilizamos as listas. Normalmente, utiliza-se a tag "ul", porém, outros tipos de listas, como "ol", podem ser utilizadas.
Por vezes, alguns programadores inserem as "ul" dentro de alguma "div". Essa prática não é necessária, pois o intuito do programador é deixar o código o mais limpo possível. Uma "ul" é, em si mesma, um elemento em bloco, portanto, não há a necessidade de inserí-lo em uma "div". Nas primeiras aulas de XHTML, aprendemos que não é recomendável inserir elementos em bloco dentro de outros elementos em bloco.
Para conseguirmos dar o efeito necessário ao menu, utilizamos JavaScript. A biblioteca JQuery é a mais utilizada para esse tipo de efeito.
