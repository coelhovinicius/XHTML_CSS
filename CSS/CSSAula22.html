Layout - Estruturação

Introdução

Nas aulas anteriores, aprendemos que as "div" são utilizadas para estruturar os grandes blocos de informações a serem inseridas em uma página.
O ideal é estruturar o layout de maneira que contenha todas as áreas necessárias. Abaixo, vemos um exemplo de sintaxe para estruturação de uma página básica: 

<div id="container">
	<div id="header"></div>
	<div id="left"></div>
	<div id="content"></div>
	<div id="right"></div>
	<div id="footer"></div>
</div>

Como vemos, uma estrutura básica compõe-se das "áreas" Cabeçalho, Esquerda, Centro, Direita e Rodapé. Devemos utilizar as IDs da melhor maneira possível, para que a página seja de fácil manutenção.
No exemplo de estruturação abaixo, estamos definindo IDs "genéricas", porém, ao se desenvolver uma página, devemos utilizar IDs mais específicas.
Por exemplo, se temos uma página com diversas "divisões". Podemos, para cada divisão da página, desenvolver DIVs que contenham, cada uma, seus elementos Cabeçalho, Esquerda, Centro, Direita e Rodapé. Assim, devemos nomear cada um desses itens diferentemente. Por exemplo, "MenuHeader", "AdsMenu", "MainMenu", "MainContent", etc.. Sempre devemos definir se iremos utilizar nomenclaturas em inglês ou português. O mais indicado, obviamente, é utilizar o inglês.

Os Layouts podem ser classificados segundo dois critérios:
Número de colunas, onde muitos sites têm colunas "padrão", onde são inseridos os dados; e
Largura da Página, que pode ser Fixa, onde a largura não varia com o tamanho da janela do browser; Líquida, que são larguras que variam em relação ao tamanho do browser, usando percentagens; e larguras Elásticas, que define a largura em relação ao tamanho de fonte, por exemplo.
Podemos utilizar uma largura Fixa, com a vantagem de se ter restrito controle de como as informações serão exibidas em quaisquer tamanhos de janelas, poré, a desvantagem é que aparecerão as barras de rolagem para conseguir ler a página em uma janela pequena, por exemplo, além do mau aproveitamento da área disponível horizontalmente se a janela do browser for muito grande, por exemplo.
Veja abaixo a sintaxe em: 

#container{
	border:1px solid black;
	width:500px;
}

Com a largura Líquida, podemos definir percentuais nos quais as estruturas da página serão exibidas na tela. Utilizamos "width:80%;". Podemos definir, também, uma largura mínima para um layout.
Abaixo, vemos outro exemplo de sintaxe: 

#container{
	border:1px solid black;
	width:/*500px*/80%;
	min-width:300px;	
}

Com o layout Elástico, conseguimos definir a largura com base no tamanho da fonte adotada, por exemplo. 

*{
	font-size:12px;
}
#container{
	border:1px solid black;
	width:30em;
}

Centralização do Layout

Podemos definir margens através da propriedade "margin". Para centralizarmos uma página, utilizamos o comando "auto" para os lados direito e esquerdo na atribuição da "margin". 

#container{
	border:1px solid black;
	width:90%;

	margin:0px auto; - "0px" para cima e para baixo e "auto" para os lados direito e esquerdo
}
