Posicionamento com CSS

Há três tipos de posicionamento em CSS:

  • Normal
  • Float
  • Absoluto

Posicionamento NORMAL

O esquema de posicionamento Normal é o esquema padrão dos browsers. É como os elementos de uma página são exibidos por padrão.

Propriedade POSITION

Muito utilizada para a estilização de páginas com CSS, a propriedade Position (position:) possibilita a customização do posicionamento dos elementos do XHTML.
Exemplo:

Positionamento Relativo

span{
	border:1px solid black;
	
	position:relative;
	left:6px;
	top:6px;
}
