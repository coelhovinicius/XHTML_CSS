CABEÇALHOS

Como estudado em XHTML, temos vários elementos que compõem a página. Os Cabeçalhos (headers) são as tags "h1, h2, etc.", que podemos estilizar de diversas maneiras. Abaixo, um exemplo de estilização: 

h1{
	border-top:1px dotted black;
	border-bottom:1px dotted black;
	border-left:10px solid black;
	background-color:#FFFF00;
	padding-left:50px;
	
	background-image:url(../img/exemplo.png) ;
	background-repeat:no-repeat;
	background-position:5px;
}
