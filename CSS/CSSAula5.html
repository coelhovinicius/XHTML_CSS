CSS - Seletores Compostos

Seletor Descendente

O seletor Descendente é aplicado a qualquer elemento que esteja dentro do elemento especificado, como visto abaixo:

EXEMPLO DE SELETOR DESCENDENTE

SELETOR IRMÃO-ADJACENTE

OUTRO EXEMPLO

				EXEMPLO DE SELETOR DESCENDENTE
				
				div em {
					color:red;
				}

Seletor Filho

O seletor Filho é aplicado a qualquer elemento que esteja dentro do elemento especificado, mesmo que esteja dentro de outro elemento, como visto abaixo:

EXEMPLO DE SELETOR FILHO 

				EXEMPLO DE SELETOR FILHO
				
				div > em {
					text-decoration:underline;
				}

Seletor Irmão-Adjacente

O seletor Irmão-Adjacente a elementos próximos. Por exemplo, se temos duas Tags "p", podemos customizar, apenas, a segunda Tag "p". Se temos um "h4" após um "p", podemos customizá-lo, também. Abaixo, o exemplo da sintaxe:

EXEMPLO DE SELETOR IRMÃO-ADJACENTE

EXEMPLO DE SELETOR IRMÃO-ADJACENTE 

			
				p + p {
					color:blue;
				}

OUTRO EXEMPLO

			
				p + h4 {
					color:grey;
				}
