Estilização de Textos - Decoração

Text Decoration

Essa propriedade possibilita decorar o texto a ser customizado. Podem ser:

  • Underline - text-decoration:underline; - aplica uma linha abaixo do texto
  • Overline - text-decoration:overline; - aplica uma linha acima do texto
  • Line-through - text-decoration:line-through; - aplica uma linha sobre o texto
  • Blink - text-decoration:blink; - o text pisca na tela;
				p{
					text-decoration:underline;
				}
				h1{
					text-decoration:overline;
				}
				h2{
					text-decoration:lign-through;
				}
				span{
					text-decoration:blink;
				}

Text Transform

A propriedade em questão controla os efeitos de capitalização do texto - caixa-alta ou caixa-baixa, podendo ser:

  • Capitalize - text-transform:capitalize; - coloca o primeiro caractere das palavras em caixa alta
  • Uppercase - text-transform:uppercase; - coloca todos os caracteres do elemento em caixa-alta;
  • Lowercase - text-transform:lowercase; - coloca todos os caracteres do elemento em caixa-baixa
