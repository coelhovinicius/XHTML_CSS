ESTILIZAÇÃO DE LINKS

A estilização de Links se dá através da utilização das pseudo-classes.
Há quatro pseudo-classes para links:

  • a:link - estiliza o link no estado inicial
  • a:visited - estiliza um link já visitado
  • a:hover - estiliza o link quando passamos o ponteiro do mouse sobre ele
  • a:active - estiliza o link quando clicamos sobre ele

Exemplos de Customização de Links

LINK EXTERNO - Direciona a uma página externa
LINK DE E-MAIL - Direciona um e-mail
LINK INTERNO - Direciona a um documento interno
LINK EM IMAGEM:

Abaixo, temos um exemplo de customização: 

a{
	text-decoration:none;
	color:#000000;
	background-color:#DDDDDD;
	border:1px solid #000000;
	width:125px;
	display:block; /*transforma um elemento inline em elemento em bloco*/
	padding:2px;
}
a:hover{
	text-decoration:underline;
	color:#DDDDDD;
	background-color:#000000;
}

Podemos, ainda, atribuir imagens aos links.
Para atribuirmos imagens, podemos criar um estilo, por exemplo, para links que contenham "http:" na referência "href", utilizando a sintaxe a[href^="http:"]: 

a[href^="http:"]{
	background:url(../img/externo.png) no-repeat right top;
	padding-right:15px;
}
a[href^="mailto:"]{
	background:url(../img/email.gif) no-repeat right top;
	padding-right:17px;
}

Se quisermos customizar um "href" que termine com um determinado tipo de extenção de arquivo, como ".jpg", por exemplo, utilizamos o "$" ao invés do "^", como visto abaixo: 

a[href$=".jpg"]{
	background:url(../img/interno.gif) no-repeat right top;
	padding-right:25px;
}

Para customizarmos links em imagens, utilizamos a sintaxe abaixo na folha de estilos CSS: 

a img{
	border:none;
}

Neste exemplo, estamos retirando a borda da imagem-link que está apresentada na página, porém, apenas as imagens (img) que estão dentro da referência âncora (a href) serão customizadas.
