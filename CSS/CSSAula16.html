ESTILIZAÇÃO DE LISTAS

As listas são muito utilizadas para a construção de menus nas páginas XHTML. A sintaxe correta para a criação de menus, sejam verticais, horizontais ou de subníveis, é com a adoção da estrutura de listas. É muito importante saber manusear as listas de uma página.

Veja abaixo um exemplo de lista:

  • GOOGLE
  • YAHOO
  • FACEBOOK
  • CEF

Podemos alterar o ícone padrão das listas, que é uma bolinha preta (disc). Há diversos tipos de customização, como visto nos exemplos abaixo:

  • list-style-type:none;
  • list-style-type:circle;
  • list-style-type:square;
  • list-style-type:disc;
  • list-style-type:lower-roman
  • list-style-type:upper-roman
  • list-style-type:georgian;
  • list-style-type:armenian;
  • list-style-type:lower-latin; ou list-style-type:lower-alpha;
  • list-style-type:upper-latin; ou list-style-type:upper-alpha;
  • list-style-type:lower-greek;
ul{
	list-style-type:circle;
}

OU

ul{
	list-style-type:square;
}

OU

ul{
	list-style-type:disc; - padrão
}

Ainda, podemos inserir imagens ao invés dos itens vistos acima. Utilizamos a seguinte sitaxe: 

ul{
	list-style-image:url(img/lista.png)
}

Há os recursos "list-style-position:inside;" ou "list-style-position:outside (padrão)" 

ul{
	list-style-position:inside; /*ou outside*/
}

Simplificação

A simplificação dos códigos acima podem ser feitas com a seguinte sintaxe, por exemplo: 

ul{
	list-style:none inside url(../img/lista.png);
}
