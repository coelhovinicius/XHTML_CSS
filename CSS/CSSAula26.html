CSS Sprite

Sprite é uma técnica em CSS utilizada para maximizar a performance das páginas XHTML. Consiste na inclusão de imagens sem uma grande quantidade de requisições ao servidor.
Abaixo, temos uma imagem inteira, dentro da qual há diversas imagens diferentes. Com o Sprite, podemos exibir, apenas, um único ícone dentre todos os que estão nessa imagem, por exemplo. Portanto, essa técnica nos possibilida que exibamos, apenas, "pedaços" de uma imagem maior. Assim, não é necessário que tenhamos um arquivo para cada imagem e, assim, diminuem-se as requisições ao servidor, aumentando a performance da página.

Abaixo há um exemplo de exibição de um dos ícones acima com Sprite:

Conseguimos esse efeito criando-se uma <div> e atribuindo uma imagem a ela com CSS. Abaixo, veja a sintaxe: 

Para XHTML: <div class="email"></div>

Para o CSS:
.email{
	background:url(../img/sprite.png) -307px -160px;
	height:25px;
	width:25px;
}
