CSS - Folhas de Estilo

A vantagem de utilizar o CSS é a maior facilidade de localização das folhas de estilo. Pode-se centralizar todos os estilos em um único lugar.

Os códigos CSS sempre devem estar no cabeçalho do documento (head).

Podemos criar um código CSS em arquivos diferentes do XHTML. Gravamos com extensão .css, obviamente.
Chamamos a folha de estilo dentro do código XHTML de duas maneiras:

				<link rel="stylesheet" type="text/css" href="css/CSSAula3Estilo.css" media="all" />
			ou	
				<style type="text/css">
					@import url("css/CSSAula3Estilo.css") screen, projection; - tipo de mídia
				</style>
