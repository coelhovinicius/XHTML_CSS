UNIDADES DE MEDIDAS

O CSS possui diversos tipos de medidas para trabalhar com a customização da página.

Medidas Absolutas

Medidas Absolutas são valores que não dependem de um valor de referência. Há cinco tipos de unidades de Medidas Absolutas:

  • Polegada - in - Exemplo: 5in
  • Centímetros - cm - Exemplo: 5cm
  • Milímetros - mm - Exemplo: 5mm
  • Pontos - pt - Exemplo: 5pt
  • Pics - pc - Exemplo: 5pc

As medidas absolutas só devem ser utilizadas quando conhecemos com detalhes as características físicas e as configurações do equipamento e do programa que rodarão o arquivo XHTML. Utilizando medidas absolutas em letras, por exemplo, elas ficarão do mesmo tamanho em telas grandes ou pequenas. Portanto, poderão parecer muito grandes em telas pequenas e muito pequenas em telas grandes, por exemplo.
Praticamente, utiliza-se, apenas, a Medida Absoluta de Pontos (pt), pois é utilizada em mídias de impressão.

Medidas Relativas

Há três tipos de medidas relativas:

  • em - refere-se a "x" vezes o valor de uma fonte definida para o mesmo elemento. Exemplo: 5em
  • ex - é igual a altura da letra "x"
  • px - dimensões fixas - calculada em relação ao dispositivo no qual sendo rodado o XHTML

Medida relativa "em" - Exemplo:

				h1{
					font-size:50px;
				}
				h2{
					font-size:20px;
				}	
				h1,h2{
					margin-left:1em; /*Aplica a medida relativa à fonte para a margem esquerda de "h1" e "h2"*/
				}

Podemos fazer uma relação entre as fontes da página inteira. Para isso, utilizamos o seletor universal "*", como no exemplo abaixo: 

				*{
					font-size:10px;
				}
				h1{
					font-size:3em;
				}
				h2{
					font-size:2em;
				}

Medidas Percentuais

Podemos utilizar Medidas Percentuais para aplicarmos aos elementos: 

				p{
					font-size:400%
				}
