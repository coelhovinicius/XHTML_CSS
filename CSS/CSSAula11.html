Estilização de Textos - Espaçamento e Alinhamento

Letter Spacing

A propriedade Letter Spacing - "letter-spacing:" - permite a customização do espaçamento entre as letras do texto de um elemento. Podemos atribuir qualquer valor, inclusive números fracionados ou negativos.
Exemplo de sintaxe: 

				p{
					letter-spacing:-5.1px;
				}

Word Spacing

A propriedade Word Spacing - "word-spacing:" - permite a customização do espaçamento entre as palavras do texto de um elemento.
Exemplo de sintaxe: 

				p{
					word-spacing:5px;
				}

Line Height

A propriedade Line Spacing - "line-height:" - permite a customização do espaçamento entre as linhas do texto de um elemento. Podemos aplicar quaisquer medidas para indicar o tamanho do espaçamento, inclusive, podemos utilizar Medidas Relativas, onde atribuímos um valor em número ou porcentagem. a base de cálculo será o tamanho da fonte do elemento ao qual está sendo aplicado o código CSS. Não aceita valores negativos.
Exemplos de sintaxe: 

				p{
					line-height:5px;
				}
				h1{
					line-height:3; - espaçamento entre linhas de três vezes o tamanho da fonte
				}
				h2{
					line-height:200%; - espaçamento entre linhas de 200% o tamanho da fonte
				}

Vertical Align

Controla o posicionamento vertical dos boxes inline.
Por exemplo, quando temos um box span em um texto, podemos atribuir customizações a esse trecho.
Temos quatro tipos de efeitos para o Vertical Align:

  • super - deixa o trecho ligeiramente acima do resto do texto
  • sub - deixa o trecho ligeiramente abaixo do resto do texto
  • text-top - move o texto ao redor do trecho ligeiramente para cima
  • text-bottom - move o texto ao redor do trecho ligeiramente para baixo

Exemplo de sintaxe: 

				span{
					vertical-align:super;
				}

Text Align

Propriedade que possibilita a customização do alinhamento do texto. Podem ser:

  • text-align:left; - alinha à esquerda
  • text-align:right; - alinha à direita
  • text-align:center; - centraliza
  • text-align:justify; - justifica o texto

Text Indent

Atribui a Indentação (recuo) apenas à primeira linha do texto do elemento a ser customizado. Podemos usar valores negativos, também. 

				text-indent:50px;
