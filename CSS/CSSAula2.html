CSS - Estilos incorporados

Com o CSS, utlizamos a Tag "style" para formarmos um "bloco" de formatação. Assim, podemos atribuir essa formatação a qualquer trecho do código XHTML. Isso diminui muito o código e facilita a manutenção da página.
A Tag Style necessita de um "tipo" sempre. Por exemplo, .style type="text/css"..
Dentro da Tag Style, colocamos as definições que desejamos.
Sempre temos que ter o Seletor que, no caso abaixo, é o "p", e as declarações, entre chaves.

				.style type="text/css".
				p, h1 {color:#006699;} - indica que todos os "p" e "h1" serão customizados
				.style.

Como já enfatizado no curso de XHTML, o CSS é responsável por toda a parte de customização da página. Assim, o XHTML não deve ser usado para fins de customização, apenas para a estrutura semântica da página, como, por exemplo, indicar o nome do documento, título, subtítulo, parágrafos, listas, hyperlinks, imagens, etc..
