ESTILIZAÇÃO DE FONTES

Para começar a aprender a estilizar as fontes, precisamos entendê-las primeiro.
Existem fontes Seguras, que são as fontes que vêm instaladas na maioria dos sistemas operacionais, fontes Genéricas, que são utilizadas como última opção na atribuição de fonte ao elemento na folha de estilo CSS, e outros tipos de fontes, que podem ser quaisquer fontes existentes. Porém, pode ser que o sistema operacional não possua a fonte existente no código e, assim, aplique a fonte padrão.

Fontes Seguras

As fontes Seguras são as fontes mais usadas, por padrão, pela maioria, senão de todos os sistemas operacionais. São elas:

  • Arial
  • Arial Black - apesar de não funcionar, pelo menos, não no meu sistema operacional e meu browser
  • Comic Sans MS
  • Courier New
  • Impact
  • Times New Roman
  • Trebuchet MS
  • Verdana

Para a folha de estilo CSS, definimos a familia de fontes que iremos aplicar: 

				p{
					font-family:"lucida grande", "lucida sans", verdana, sans-serif;
				}

No exemplo acima, o browser irá utilizar a fonte informada na linha do código CSS. Como há diversas fontes informadas, ele irá verificar se ele possui as fontes no seu sistema operacional, sempre seguindo a ordem da primeira até a última. Caso ele não possua a primeira fonte, "lucida grande", disponível no sistema operacional, ele irá partir para a segunda fonte, "lucida sans" e, assim, sucessivamente. Por isso, é importante que a última fonte informada no código CSS seja a fonte mais genérica possível, uma que, com certeza, todos os sistemas operacionais tenham. Normalmente, trata-se da fonte "sans-serif".
Nesse link, verificamos as fontes seguras definidas pela W3 Schools.

Fontes Genéricas

O W3C define cinco Grupos de fontes Genéricas, destinadas à última opção, caso nenhuma das anteriores esteja disponível no sistema operacional residente:

  • Serif
  • Sans-Serif
  • Cursive
  • Fantasy
  • Monospace

Cada Grupo de fontes Genéricas possui diversos tipos de fontes, que pode ser verificado no link da W3 Schools.

Estilos de Fontes

Propriedade "font-style", que permite definir que as fontes sejam exibidas em:

  • Itálico - "font-style:italic;"
  • Normal - "font-style:normal;"
  • Oblíquo - "font-style:oblique;"
  • Herança - "font-style:inherit;"

Peso da Fonte

Propriedade "font-weight", define a tonalidade da fonte, baseada no parâmetro "peso". Podemos utilizar os valores:

  • Bold - deixa a letra em Negrito - "font-weight:bold;"
  • Lighter - deixa a letra um pouco (quase nada) mais Fina - "font-weight:lighter;"
  • Valores que variam de 100 - "font-weight:100;" a 900 - "font-weight:900;"

Propriedade Font Variant

Definida pela sintaxe "font-variant:small-caps", Transforma as letras minúsculas de um texto em letras maiúsculas, porém, um pouco menores que as letras maiúsculas normais do texto.

Propriedade Font Size

Essa propriedade já foi muito aplicada nas aulas anteriores, onde podemos alterar o tamanho das letras.

Exemplo

Abaixo, podemos verificar um exemplo de sintaxe com as propriedades de fonte que foram vistas nessa aula: 

				p{
					font-style:italic;
					font-variant:small-caps;
					font-weight:bold;
					font-size:6px;
					font-family:"lucida grande", "lucida sans", verdana, sans-serif;
				}

Simplificação das declarações de fontes

Como em todos os recursos CSS que utilizamos anteriormente, há uma maneira de simplificar a declaração dos valores, também, para as Fontes, conforme o exemplo abaixo: 

				p{
					font:italic small-caps bold 10px/150% verdana, sans-serif;
				}
