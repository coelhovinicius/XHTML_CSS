CSS - Seletores Simples

Como dito anteriormente, o seletor é um dos elementos básicos do CSS.
No exemplo abaixo, vemos o seletor "p", que seleciona as entidades "p" para utilizarem o CSS codificado: 

				p {
					color:#006699;
					font-family:"Comic Sans MS";
				}
Temos três tipos de seletores:
  • Simples - Constituído de um só elemento, podendo ser associado a um ID ou a uma pseudo-classe
  • Compostos -
  • Pseudo-seletores -


Seletor SIMPLES


Seletor Simples UNIVERSAL

Seletor aplicado a todos os elementos do documento. É representado através do símbolo asterísco "*".
			* {
				margin:0; - remove a margem padrão dos browsers, em todos os elementos
			}

Seletor Simples ELEMENTO

Seletor aplicado ao elemento, como já estudado anteriormente. No caso abaixo, aplicamos ao elemento "p" - parágrafo.
				p {
					color:#006699;
					font-family:"Comic Sans MS";
				}

Seletor Simples CLASSE

Com esse seletor, podemos aplicar definições pontuais ao nosso código. Por exemplo, abaixo temos o elemento "p" com uma classe que denominamos de "destaque", podendo ser qualquer outro nome desejado. Isso cria um indicador, que irá puxar especificamente as definições para aquele elemento "p" em questão. Qualquer outro elemento pode receber uma classe, para que possa ter uma definição CSS diferente dos demais elementos.
 
				STRING XHTML PARA IDENTIFICAR A CLASSE PARA O ELEMENTO QUE IRÁ LEVAR AS CONFIGURAÇÕES CSS:
				
				p class="destaque" - utilizam-se as Tags  para abrir e fechar: <>
				
				
				STRING CSS PARA CONFIGURAR AS DEFINÇÕES DO TRECHO:
				
				.destaque {
				color:#FFD700;
				font-family:"arial";
				}


Assim, esse texto está aparecendo diferente de todos os outros parágrafos da página.

Seletor Simples ID

Por exemplo, quando utilizamos DIVs, também, podemos identificá-las com IDs, como já visto nas aulas de XHTML. Assim, podemos atribuir diversas definições CSS diferentes a esses IDs, como visto abaixo:
					STRING XHTML PARA IDENTIFICAR O ID PARA A DIV QUE IRÁ LEVAR AS CONFIGURAÇÕES CSS:
					
					div id="exemplo" - utilizam-se as Tags  para abrir e fechar: <>
					
					
					STRING CSS PARA CONFIGURAR AS DEFINÇÕES DO TRECHO:
					
					#exemplo
					{
					background-color:#7FFFD4;
					}

Seletor Simples ATRIBUTO

Seletor que "casa" com um determinado atributo de um de nossos elementos Por exemplo, quando há uma marcação no XHTML do tipo inline EM, que dá ênfase ao trecho indicado, podemos atribuir definições CSS especificamente para esses elementos também.
				STRING XHTML PARA IDENTIFICAR O TÍTULO DA EM RELACIONADA:
					
				em title="trecho" - utilizam-se as Tags  para abrir e fechar: <>
					
				
				STRING CSS PARA CONFIGURAR AS DEFINÇÕES DO TRECHO:
					
				em[title=trecho] {
				color:#00008B;
				background-color:#EEDD82;
				}

Ainda temos outro exemplo de utilização. Por exemplo, se quizermos identificar qualquer parte do texto que tenha título contendo a palavra "exemplo", desde que não seja separado por hífen, utilizamos a sintaxe abaixo:
			
				em[title~=exemplo] {
				color:#00BFFF;
				font-family:"arial";
				}

Indicamos acima que os textos com hífen não são considerados com a utilização daquela sintaxe. Portanto, para casos com hífen, temos a sintaxe abaixo:
			
				em[title|=outro] {
				color:#CDCDC1;
				font-family:"Lucida Console";
				}
